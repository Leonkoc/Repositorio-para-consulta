Acessar dados de matriz com índices
Podemos acessar os dados dentro de arrays usando índices.

Os índices de matriz são escritos na mesma notação de colchetes que as strings usam, exceto que, em vez de 
especificar um caractere, eles especificam uma entrada na matriz. Assim como as strings, os arrays usam 
indexação baseada em zero, então o primeiro elemento em um array tem um índice de 0.

Exemplo

const array = [50, 60, 70];
console.log(array[0]);
const data = array[1];
As console.log(array[0])impressões 50, e datatem o valor 60.
